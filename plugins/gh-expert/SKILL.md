# GitHub Expert Plugin

This plugin provides a suite of advanced GitHub CLI skills for autonomous agents to use when managing pull requests, issues, and workflow.

## Available Skills

- **gh-pr**: Automation for creating, reviewing, and managing pull requests via `gh`.

## Usage

Use the specific skills when you need advanced github operations via the GitHub CLI.
Example: `/gh-pr create` to trigger a PR creation workflow.
Read https://cli.github.com/manual/ for documentation on `gh`.
