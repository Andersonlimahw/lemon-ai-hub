# Closed Testing Checklist

> Google Play may require closed testing (20+ testers for personal accounts on certain app types) before production.

- [ ] Tester group created (email list)
- [ ] Release eligible for closed track
- [ ] AAB uploaded to closed testing track
- [ ] Testers accepted the opt-in link
- [ ] Main journeys verified by testers
- [ ] Device coverage (min + target SDK, varied screens)
- [ ] No policy issues surfaced
- [ ] Crashes / ANRs reviewed and resolved
- [ ] Pre-launch report reviewed
- [ ] Minimum required testers participated for required duration
- [ ] **Human decides** staged rollout (agent never auto-advances)
