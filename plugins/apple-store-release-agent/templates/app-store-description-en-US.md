## Description
Organize amateur football (soccer) matches with ease. Create games, invite friends, build teams, and keep everything in one place.

With Vai ter fut you can:
• Create matches and set date, time, and location
• Invite players and confirm attendance
• Auto-build balanced teams
• Track invites and each player's status
• Get reminders so you never miss a game

Built for everyone who loves a pickup game and wants to settle the lineup in seconds.

## Soccer Coins (virtual currency)
The app offers Soccer Coins, a virtual currency used only inside the app to unlock features and customization. Soccer Coins **have no monetary value, cannot be converted into cash, and cannot be withdrawn**.
